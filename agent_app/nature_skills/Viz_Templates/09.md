这三张图展示了数学建模论文中常见的三类可视化需求：**复杂网络图**、**带局部放大的统计图**、以及**统计饼图**。

以下是针对每一张图的**推荐工具**和**制作方法**：

---

### 图1：复杂网络关系图 (Figure 4: Sub-network of Musical Influence)

这张图展示了节点（音乐家）之间的连线（影响关系），节点颜色代表流派，大小代表影响力。

**推荐工具：Gephi (首选) 或 Python (NetworkX)**

#### 方法一：使用 Gephi (最推荐，效果最炫酷)
Gephi 是一款专门用于网络分析和可视化的开源软件，非常适合处理这种复杂的连线图。
1.  **数据准备**：准备两个 CSV 文件。
    *   `Nodes.csv`: 包含 `Id`, `Label` (名字), `Category` (流派/颜色), `Size` (影响力数值)。
    *   `Edges.csv`: 包含 `Source`, `Target`, `Weight` (如果有权重)。
2.  **导入 Gephi**：通过 "Data Laboratory" 导入上述文件。
3.  **布局 (Layout)**：在 "Overview" 界面，左下角选择布局算法。这张图看起来使用的是 **ForceAtlas 2** 或 **Fruchterman Reingold** 算法，它们能让相互连接的节点聚在一起，互不连接的分开。
4.  **外观 (Appearance)**：
    *   **节点大小**：选择 `Ranking` -> `Size` -> 根据影响力数值（如 Degree）调整。
    *   **节点颜色**：选择 `Partition` -> `Color` -> 根据流派（Category）自动上色。
5.  **标签 (Labels)**：开启底部的黑色 `T` 图标显示标签，调节字体大小。
6.  **预览与导出**：去 "Preview" 界面，调整线条弯曲度 (Curved)，导出为 PDF 或 PNG。

#### 方法二：使用 Python (NetworkX)
如果你习惯写代码，可以用 Python。
```python
import networkx as nx
import matplotlib.pyplot as plt

# 建立图
G = nx.Graph() # 或 DiGraph
G.add_edges_from([('Beatles', 'Bob Dylan'), ...])

# 布局
pos = nx.spring_layout(G, k=0.5) # 类似力导向布局

# 绘图
nx.draw_networkx_nodes(G, pos, node_size=..., node_color=...)
nx.draw_networkx_edges(G, pos, connectionstyle="arc3,rad=0.1") # 弯曲连线
nx.draw_networkx_labels(G, pos)
plt.show()
```

### 图2：带局部放大的分布拟合图 (Figure 3: Frequency Distribution)

这张图展示了数据的频率分布（蓝线）和幂律分布拟合（红虚线），最关键的特征是**图中有一个放大的子图 (Inset Plot)**。

**推荐工具：MATLAB 或 Python (Matplotlib)**

#### 方法一：使用 Python (Matplotlib)
Python 的 `mpl_toolkits` 库非常擅长处理这种“图中图”。
```python
import matplotlib.pyplot as plt
from mpl_toolkits.axes_grid1.inset_locator import inset_axes, mark_inset

fig, ax = plt.subplots()

# 1. 绘制主图
ax.plot(x, y_data, label='Frequency distribution')
ax.plot(x, y_fit, '--', label='Fitting power-law')
ax.legend()

# 2. 创建嵌入的子坐标轴 (局部放大图)
# loc=10 表示位置，width/height 是比例
axins = inset_axes(ax, width="40%", height="30%", loc='center') 

# 3. 在子图中再画一遍数据
axins.plot(x, y_data)
axins.plot(x, y_fit, '--')

# 4. 设置子图的显示范围 (Zoom in)
axins.set_xlim(0.04, 0.15) # 设置你想放大的 X 轴范围
axins.set_ylim(0, 0.15)    # 设置你想放大的 Y 轴范围

# 5. 添加连线 (把子图和主图对应区域连起来)
mark_inset(ax, axins, loc1=2, loc2=4, fc="none", ec="0.5")

plt.show()
```

#### 方法二：使用 MATLAB
MATLAB 在数模比赛中非常常用，它的 `axes` 命令可以手动在图上再画一个坐标系。
```matlab
% 绘制主图
figure;
plot(x, y, 'b-'); hold on;
plot(x, y_fit, 'r--');

% 插入子图
% axes('Position',[left bottom width height]) 归一化坐标(0-1)
axes('Position',[.4 .4 .3 .3]); 
box on; % 加边框
plot(x, y, 'b-'); hold on;
plot(x, y_fit, 'r--');
xlim([0.04 0.15]); % 设置局部范围
```

### 图3：分离式饼图 (Figure 6: Statistics of Musicians)

这张图是两个饼图，且部分扇区被“炸开”（Exploded）以示强调，带有阴影效果。

**推荐工具：Excel (最简单快捷) 或 Python**

#### 方法一：使用 Excel (推荐)
这种图在商业报表和论文中很常见，Excel 做这个最快且效果足够好。
1.  **输入数据**：两列数据，一列是流派，一列是数量。
2.  **插入图表**：选择数据 -> 插入 -> 饼图 -> **三维饼图** 或 **二维饼图**。
3.  **分离扇区**：
    *   点击饼图，**双击**你想要分离的那一块扇区（确保只选中了那一块）。
    *   按住鼠标左键向外拖拽，即可实现“分离”效果。
4.  **美化**：
    *   右键添加数据标签 (Data Labels)。
    *   调整颜色配色（建议使用 ColorBrewer 等专业配色）。

#### 方法二：使用 Python (Matplotlib)
```python
import matplotlib.pyplot as plt

labels = ['Folk', 'Blues', 'Pop/Rock', ...]
sizes = [81, 85, 470, ...]
# explode 数组对应每个扇区，0表示不分离，0.1表示分离距离
explode = (0.1, 0.1, 0, 0, ...) 

plt.pie(sizes, explode=explode, labels=labels, autopct='%1.1f%%',
        shadow=True, startangle=90) # shadow=True 开启阴影
plt.axis('equal')
plt.show()
```
