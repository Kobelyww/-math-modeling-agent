# 数学建模常用方法与技巧

## 一、优化类模型

### 线性规划（Linear Programming）
- **适用场景**：目标函数和约束条件均为线性的优化问题，如资源分配、生产计划
- **标准形式**：min c^T x  s.t. Ax ≤ b, x ≥ 0
- **求解工具**：Python pulp / scipy.optimize.linprog / gurobi
- **关键点**：决策变量定义、约束完整性、灵敏度分析

### 整数规划（Integer Programming）
- **适用场景**：决策变量必须取整数的场景，如选址问题、排班问题
- **分支**：0-1整数规划、混合整数规划（MILP）
- **求解方法**：分支定界法、割平面法、隐枚举法
- **注意**：NP-hard问题，大规模需要启发式算法

### 非线性规划（Nonlinear Programming）
- **适用场景**：目标函数或约束包含非线性项
- **求解方法**：拉格朗日乘子法、KKT条件、罚函数法、SQP
- **工具**：scipy.optimize.minimize, cvxopt

### 多目标优化
- **核心概念**：Pareto最优解、非支配排序
- **常用方法**：
  - 线性加权法：将多目标加权为单目标
  - ε-约束法：保留一个目标，其余转为约束
  - 目标规划法：设定各目标期望值，最小化偏差
  - NSGA-II：精英非支配排序遗传算法
- **权重确定**：AHP层次分析法、熵权法、TOPSIS

### 动态规划（Dynamic Programming）
- **核心思想**：最优子结构 + 重叠子问题
- **适用场景**：多阶段决策问题、背包问题、路径规划、资源分配
- **关键步骤**：阶段划分 → 状态定义 → 决策选择 → 状态转移方程 → 边界条件
- **维数灾难**：状态空间过大时的应对策略（近似DP、强化学习）

## 二、评价与决策模型

### 层次分析法（AHP）
- **步骤**：
  1. 建立递阶层次结构（目标层→准则层→方案层）
  2. 构造两两比较判断矩阵（1-9标度法，Saaty）
  3. 计算权重向量（特征向量法 / 和积法 / 方根法）
  4. 一致性检验（CR = CI/RI < 0.1）
  5. 层次总排序
- **关键点**：判断矩阵需通过一致性检验，否则需要重新打分
- **改进**：模糊AHP（处理不确定判断）、网络层次分析法（ANP，处理依赖关系）

### TOPSIS法
- **核心思想**：最优方案应距离正理想解最近、距离负理想解最远
- **步骤**：
  1. 指标正向化（极小型→极大型、中间型→极大型、区间型→极大型）
  2. 数据标准化（向量归一化 / min-max标准化）
  3. 确定正理想解Z+和负理想解Z-
  4. 计算各方案与正负理想解的距离
  5. 计算相对贴近度并排序
- **优点**：对数据分布无要求，计算简单直观

### 熵权法
- **核心思想**：信息熵越小→指标变异程度越大→权重越大
- **步骤**：数据标准化 → 计算比重矩阵 → 计算熵值 → 计算差异系数 → 归一化得权重
- **常结合**：与TOPSIS结合形成熵权-TOPSIS法

### 模糊综合评价
- **步骤**：确定因素集和评语集 → 构建隶属度矩阵 → 确定权重向量 → 模糊合成 → 结果分析
- **隶属函数选择**：三角形、梯形、高斯型
- **合成算子**：M(∧,∨) / M(·,∨) / M(·,+)

### 数据包络分析（DEA）
- **适用场景**：多投入多产出的效率评价（不需预设权重和函数形式）
- **经典模型**：CCR模型（CRS假设）、BCC模型（VRS假设）
- **结果分析**：综合效率 = 纯技术效率 × 规模效率

## 三、预测模型

### 时间序列分析
- **ARIMA模型**：
  - AR(p)：自回归，用历史值预测当前值
  - I(d)：差分阶数，使序列平稳化
  - MA(q)：移动平均，用历史误差修正预测
  - **建模步骤**：平稳性检验(ADF) → 差分定阶 → ACF/PACF定p,q → 模型检验 → 预测
- **SARIMA**：带季节性的ARIMA
- **GARCH**：波动率预测（金融领域）
- **指数平滑法**：简单指数平滑 / Holt双参数（带趋势）/ Holt-Winters（带趋势+季节性）

### 灰色预测 GM(1,1)
- **适用场景**：数据量少（≥4个）、信息不完全的小样本预测
- **步骤**：
  1. 级比检验（判断建模可行性，λ(k)∈(e^{-2/(n+1)}, e^{2/(n+1)})）
  2. 原始序列一次累加（1-AGO）生成
  3. 建立白化微分方程：dx/dt + ax = b
  4. 最小二乘估计参数a,b
  5. 求解时间响应函数
  6. 累减还原得预测值
  7. 精度检验（残差检验、后验差检验C和P）
- **改进**：新陈代谢GM(1,1)（滚动更新数据）

### 回归分析
- **线性回归**：OLS最小二乘 → R²检验 → F检验 → t检验
- **多元回归**：多重共线性诊断（VIF<10）、逐步回归
- **非线性回归**：曲线拟合（指数、对数、幂函数、Logistic）
- **正则化**：Ridge（L2正则）、Lasso（L1正则，可特征选择）、ElasticNet
- **逻辑回归**：二分类问题的概率预测

### 机器学习预测
- **随机森林**：集成Bagging+随机特征选择，对异常值和缺失值鲁棒
- **XGBoost / LightGBM**：梯度提升树，比赛常用，支持特征重要性分析
- **SVM回归（SVR）**：适合小样本高维数据
- **BP神经网络**：多层感知机，万能逼近定理
- **LSTM**：长短期记忆网络，适合序列预测

## 四、微分方程模型

### 常微分方程模型
- **人口增长**：Malthus（指数增长）→ Logistic（阻滞增长，有环境容量K）
- **传染病模型**：SIR模型（易感-感染-恢复）、SEIR模型（加入潜伏期）
- **药物动力学**：一室模型、二室模型
- **捕食者-猎物**：Lotka-Volterra方程
- **求解方法**：解析解（分离变量、常数变易）、数值解（RK4龙格库塔）

### 偏微分方程模型
- **热传导方程**：∂u/∂t = α∇²u（Fourier定律）
- **波动方程**：∂²u/∂t² = c²∇²u
- **扩散方程**：Fick定律，污染物扩散
- **求解方法**：分离变量法、有限差分、有限元

## 五、随机与统计模型

### 概率模型
- **蒙特卡洛模拟**：大量随机试验近似求解（π的计算、期权定价、排队系统）
- **马尔可夫链**：状态转移矩阵P、稳态概率πP=π
- **排队论**：M/M/1等模型（到达率λ，服务率μ，服务强度ρ=λ/μ）
- **贝叶斯推断**：先验概率 + 样本信息 → 后验概率

### 假设检验
- **参数检验**：t检验（均值）、F检验（方差）、χ²检验（分布拟合）
- **非参数检验**：Wilcoxon秩和检验、Kruskal-Wallis检验、K-S检验
- **显著性水平**：通常α=0.05，p<α则拒绝原假设

### 聚类分析
- **K-means**：肘部法则确定K值，对初始中心敏感
- **层次聚类**：凝聚法/分裂法，树状图可视化
- **DBSCAN**：密度聚类，可发现任意形状聚类，不需预设K
- **高斯混合模型（GMM）**：软聚类（给出属于各类的概率）

## 六、图论与网络模型

### 最短路问题
- **Dijkstra**：非负权重单源最短路（贪心，O(V²)或O(ElogV)）
- **Floyd-Warshall**：全源最短路（DP，O(V³)），可处理负权
- **应用**：交通路径规划、物流配送路线

### 网络流问题
- **最大流**：Ford-Fulkerson / Edmonds-Karp / Dinic算法
- **最小费用最大流**：在最大流基础上最小化总费用
- **应用**：运输调度、通信带宽分配

### 最小生成树
- **Prim算法**：基于节点扩展，适合稠密图
- **Kruskal算法**：基于边排序+并查集，适合稀疏图
- **应用**：通信网络设计、电网布线

### TSP与VRP问题
- **TSP（旅行商问题）**：NP-hard，精确解用分支定界，近似解用遗传算法/模拟退火/蚁群算法
- **VRP（车辆路径问题）**：TSP的扩展，加入容量、时间窗等约束
- **启发式求解**：最近邻、节约里程法、遗传算法、禁忌搜索
